# 📹 Ketentuan Video Tugas Praktikum Mobile (Java)

> **Tugas Praktikum 1 sampai 7** wajib diunggah ke YouTube sesuai ketentuan berikut.

---

## I. Ketentuan Isi Video

| No | Aspek | Keterangan |
|----|-------|------------|
| 1 | **Identitas** | Tampilkan/sebutkan **Nama** dan **NIM** di awal video. |
| 2 | **Demo Aplikasi** | Tunjukkan aplikasi berjalan di **Emulator/HP**. Jelaskan setiap fungsi tombol dan alur aplikasi secara detail. |
| 3 | **Bedah Kode (Java)** | • Jelaskan struktur dan logika utama pada file `.java` (Activity/Fragment).<br>• Jelaskan komponen utama pada file `.xml` (Layout).<br>• Pastikan menjelaskan alur data dan **Event Handling** (misal: Button Click). |
| 4 | **Kualitas** | Minimal **720p (HD)** agar baris kode terbaca jelas. |

---

## II. Format Upload YouTube

### Judul Video

```
[TUGAS MOBILE - JAVA] - [Tugas Ke-X] - Nama Lengkap - NIM
```

**Contoh:**

```
[TUGAS MOBILE - JAVA] - Tugas 1 - Mahendra Kirana - H101...
```

### Privasi

Setel sebagai **"Public"** atau **"Unlisted"** agar asisten bisa menilai.

---
